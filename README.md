# Personal Portfolio Website

A fully responsive personal portfolio website designed to showcase skills, projects, and services. This project features a modern, dark-themed UI with smooth scroll animations and interactive elements.

-   **Responsive Design:** Fully adaptable layout that works seamlessly on desktops, tablets, and mobile devices.
-   **Interactive UI:**
    -   **Typing Effect:** Dynamic text animation in the hero section using `Typed.js`.
    -   **Scroll Animations:** Elements fade in and slide into view as you scroll, powered by `ScrollReveal`.
    -   **Sticky Navigation:** The header stays fixed at the top with a glass-morphism effect on scroll.
    -   **Active Link Highlighting:** Navigation links update automatically based on the current scroll position.
-   **Sections:**
    -   **Home:** Hero section with a profile image and social media links.
    -   **About:** Detailed introduction and professional summary.
    -   **Services:** Grid layout highlighting key skills (React, Responsive Design, UI/UX).
    -   **Portfolio:** Showcase of recent projects with hover effects.
    -   **Contact:** Functional-looking contact form with email integration (mailto).
